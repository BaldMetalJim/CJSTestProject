# cesium-materials
This is a materials manager for the Cesium Test Project.  An SPA designed to manage materials for
a construction site.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
